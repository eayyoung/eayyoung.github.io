centos_文件处理命令.目录

发表于2024-03-07|centos命令，目录
|阅读量:

Ls: list(列表）

路径：/bin/ls
权限：所有用户
描述：显示目录文件
语法：ls 选项\[\-ald\] \[文件或目录\] 选项\[ \]中为可填或不填内容

-a 显示所有文件或隐藏文件>all

​-l 详细信息显示 > long -h 显示字节

-d 查看目录属性

​-i 索引结点 >inod



-rw -r–r–

- 文件类型 （- 二进制文件 d 目录 l 软连接文件)

权限

rw- r– r–
u 所有者 g 所属组 o 其他人

r 读 w写 x执行 _execute

Mkdir: make directories 建立目录

路径：/bin/mkdir

权限：所有用户

描述：创建新目录

语法：mkdir 选项[-p] [目录名]
[ ]中为可填或不填内容

-p 递归创建

案例:

cd: change directory 更改目录

路径：shell 内置命令

权限：所有用户

描述：切换目录

语法：cd [目录名]

案例：

pwd: print working directory 显示当前工作目录

路径：/bin/pwd

权限：所有用户

描述：显示当前目录

语法：pwd

案例：
$ pwd /wslsp/Japan

rmdir: remove empty directories 空白文件夹删除

路径：/bin/rmdir

权限：所有用户

描述：删除空目录

语法：rmdir [目录名]

案例：
$ rmdir /wslsp/Japan

cp: copy 复制

路径：/bin/cp

权限：所有用户

描述：复制文件或目录

语法：cp -rp [原文件或目录名] [目标目录]

-r 复制目录

-p 保留文件属性

案例：

(文件复制目录到目标中间有空格）

mv: move 改变位置，移动

路径：/bin/mv

权限：所有用户

描述：剪切文件或改名

语法：mv [原文件或目录名] [目标目录]

案例：

rm: remove 删除

路径：/bin/rm

权限：所有用户

描述：删除文件

语法：`rm -rf [文件或目录] [目录名]

-r 删除目录

-f 强制执行

案例：
$ rmdir /wslsp/Japan
