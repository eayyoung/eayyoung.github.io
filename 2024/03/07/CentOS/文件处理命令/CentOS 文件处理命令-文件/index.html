centos_文件处理命令.文件

发表于2024-03-07|centos命令文件
|阅读量:

touch: touch 创建空文件

  • 路径：/bin/touch

  • 权限：所有用户

  • 描述：创建空文件

  • 语法：touch [文件名]

  • 案例：

cat: concatenate 连锁

  • 路径：/bin/cat

  • 权限：所有用户

  • 描述：显示文件内容

  • 语法：cat [文件名]

    -n 显示行号

  • 案例：

    • $ cat /tmp/Japan/boduo
    • $ cat -n /tmp/Japan/lsp.shell

tac: 显示文件内容（反向列示）

  • 路径：/usr/bin/tac

  • 权限：所有用户

  • 描述：显示文件内容

  • 语法：`tac [文件名]

    -n 显示行号

  • 案例：

    • $ tac /tmp/Japan/boduo.shell

more: more 更多的，更大的

  • 路径：/bin/more

  • 权限：所有用户

  • 描述：分页显示文件内容

  • 语法：more [文件名]

    • （空格）或f 翻页
    • （Enter） 换行
    • q或Q 退出

  • 案例：

    • $ more /tmp/Japan/boduo.shell

less: less 更少的，较小的

  • 路径：/usr/bin/less

  • 权限：所有用户

  • 描述：分页显示文件内容（可向上翻页）

  • 语法：less [文件名]

    • （空格）或f 翻页
    • （Enter） 换行
    • （PgUp） 向上翻页
    • 上方向键 向上换行
    • q或Q 退出

  • 案例：

    • $ less /tmp/Japan/boduo.shell

head: head 头部，顶部

  • 路径：/usr/bin/head

  • 权限：所有用户

  • 描述：显示文件前几行

  • 语法：head [文件名]

    -n 指定行数

  • 案例：

    • $ head -n 20 /tmp/Japan/boduo.shell

tail: tail 尾巴，尾部

  • 路径：/usr/bin/tail

  • 权限：所有用户

  • 描述：显示文件后几行

  • 语法：tail [文件名]

    -n 指定行数

    -f 动态显示文件末尾内容

  • 案例：

    • $ tail -n 15 /tmp/Japan/boduo.shell

ln: link 链接，与…有关联

  • 路径：/bin/ln

  • 权限：所有用户

  • 描述：生成链接文件

  • 语法：ln -s [原文件] [目标文件]

    -s 创建软链接

  • 案例：

chmod: change the permissions mode of a file 修改文件的权限模式

  • 路径：/bin/chmod

  • 权限：所有用户

  • 描述：改变文件目录权限

  • 语法：chmod [ {ugao} {+-=} {rwx} ] [文件或目录]

    （u:所有者 g:所属组 a:所有人 o:其他人 r:读权 w：写权 x:执行权）
    [mode=421] [文件或目录]

    -R 递归修改

  • 案例：

    • $ chmod u+x mynote.list
      • 给所有者增加执行权
    • $ chmod g-r mynote.list
      • 去除所属组的阅读权
    • $ chmod o=rwx mynote.list
      • 给其他人所有权限

  • 权限数字表示：

    r w x
    4 2 1

    rwxrw-w– 761 rwx=4+2+1 rw=4+2 x=1

    • 案例：
    代表字符 权限 对文件含义 对目录含义
    r 读权限 可以查看文件内容 可以列出目录中的内容
    w 写权限 可以修改文件内容 可以在目录中创建和删除文件
    x 执行权限 可以执行文件 可以进入目录

chown: change file ownership 更改文件所有权

  • 路径：/bin/chown
  • 权限：所有用户
  • 描述：改变文件或目录的所有者
  • 语法：chown [选项] [所有者] [:[组]] 文件
  • 案例：
    • $ chown -R mysql:mysql /app/mysql/
      改变文件mysql的所有者所有组为mysql

chgrp: change file group ownership 更改文件组所有权

  • 路径：/bin/chgrp
  • 权限：所有用户
  • 描述：改变文件或目录的所属组
  • 语法：chgrp [用户] [文件或目录]
  • 案例：
    • $ chgrp yiyang mynote.list
      改变文件mynote.list的所有组为yiyang

umask: the user file-creation mask 用户文件创建掩码

  • 路径：shell 内置命令

  • 权限：所有用户

  • 描述：设置、显示文件的缺省(默认）权限

  • 语法：umask [-S]
    -S 以rwx形式显示新建文件缺省（默认)权限


  • 案例：

source:
在当前bash环境下读取并执行FileName中的命令

  • 路径：bash shell的内置命令
  • 权限：所有用户
  • 描述：点命令，就是个点符号
  • 语法：source [文件名]
  • 案例：$ source /etc/profile
