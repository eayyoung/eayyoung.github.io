配置 Prettier 格式化工具的完整指南

Prettier 是一个流行的代码格式化工具，支持多种语言包括 JavaScript、TypeScript、HTML、CSS、JSON 和 Markdown 等。以下是详细的配置方法：

2. 配置文件

Prettier 支持多种配置文件格式，按优先级排序：

  1. .prettierrc 文件 (JSON/YAML/JS)
  2. .prettierrc.json (明确 JSON 格式)
  3. .prettierrc.js 或 prettier.config.js (导出对象)
  4. package.json 中的 prettier 属性

4. 常用配置选项说明

选项 类型 默认值 说明
printWidth number 80 行宽限制
tabWidth number 2 缩进空格数
useTabs boolean false 使用制表符代替空格
semi boolean true 语句末尾添加分号
singleQuote boolean false 使用单引号代替双引号
trailingComma string “es5” 多行时末尾逗号 (“none”|”es5”|”all”)
bracketSpacing boolean true 对象字面量括号空格 { foo: bar }
jsxSingleQuote boolean false JSX 中使用单引号
arrowParens string “always” 箭头函数参数括号 (“always”|”avoid”)
endOfLine string “lf” 换行符 (“lf”|”crlf”|”cr”|”auto”)
proseWrap string “preserve” Markdown 文本换行 (“always”|”never”|”preserve”)
