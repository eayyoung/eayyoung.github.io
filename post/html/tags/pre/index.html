定义和用法

<pre> 标签定义预先格式化的文本。

<pre> 元素中的文本以等宽字体显示，文本保留空格和换行符。
文本将完全按照 HTML 源代码中所写的方式显示。

请参阅:

标签 描述
<code> 定义计算机程序的样例输出。
<samp> 定义键盘输入。
<kbd> 定义键盘输入。
<var> 定义变量。
