centos_网络命令&分区挂载

发表于1970-01-01|centos命令
|阅读量:

write: write 编写，写作

路径：/usr/bin/write

权限：所有用户

描述：给用户发信息 以ctrl+D结束 ctrl+Enter 回退或delete删除

语法：write [用户名]

案例：$ write yiyang

wall: write all 写给所有

路径：/usr/bin/wall

权限：所有用户

描述：发广播信息（所有人可接收） 以ctrl+D结束 ctrl+Enter 回退或delete删除

语法：wall [message：消息]

案例：$ wall wo shi ge da shuai ge!
          发送内容为‘我是大帅哥！’消息

ping: ping 发送回显消息

路径：/bin/ping

权限：所有用户

描述：测试网络连通性

语法：`ping 选项[-c] [ip地址]

-c 指定发送次数

案例：$ ping 192.168.12.22

wget: web get 网络获得

路径：内置命令

权限：所有用户

描述：进行网络文件下载

语法：wget 选项[-b] [url]
            (url:下载网址或链接）

-b 后台下载

案例：$ wget -b www.99re.net

curl: CommandLine Uniform Resource Locator

路径：/usr/bin/curl

权限：所有用户

描述：发起网络请求

语法：curl 选项[-O] [url]
            (url:下载网址或链接）

-O 用于下载使用

案例：$ curl -O www.99re.net

ifconfig: interface configure 网卡配置（centos 7为：ip addr）

路径：/sbin/ifconfig

权限：root

描述：查看和设置网卡信息

语法：ifconfig [网卡名称] [ip地址]

案例：$ ifcomfig eth0 192.168.5.34

mail: mail 邮件 （centos 7 暂不具备该命令）

路径：/bin/mail

权限：所有用户

描述：查看发送电子邮件

语法：mail [用户名]

案例：$ mail root

last: last 最后的，最近的

路径：/usr/bin/last

权限：所有用户

描述：列出目前与过去登入系统的用户信息

语法：last

案例：$ last

traceroute: traceroute 路由跟踪 （centos 7 暂不具备该命令）

路径：/bin/traceroute

权限：所有用户

描述：显示数据包到主机间的路径

语法：traceroute

案例：$ traceroute www.9u.net

netstat: netstat 查询网络数据结构与网络状态 （centos 7 暂不具备该命令）

路径：/bin/netstat

权限：所有用户

描述：显示网络相关信息

语法：netstat 选项[ ]：

-t TCP协议

-u UDP协议

-l 监听

-r 路由

-n 显示ip地址和端口号

案例：

setup: setup 安装 （centos 7 暂不具备该命令）

路径：/usr/bin/setup

权限：root

描述：配置网络

语法：setup

案例：$ setup

mount: mount 挂载命令

路径：/bin/mount

权限：所有用户

描述：挂载设备

语法：mount [-t 文件系统 ] 设备文件名 挂载点

案例：

unmount: unmount 卸载挂载命令 (需退出挂载目录卸载）

路径：/bin/unmount

权限：所有用户

描述：卸载挂载设备

语法：unmount [设备文件名] 或 [挂载点]

案例：$ unmount /dev/sr0 或 /mnt/cdrom
