npm常用命令

npm 是 Node.js 的包管理器，用于管理和操作 JavaScript 包（库）。以下是一些常用的 npm 命令：

  1. 安装包:

    • 全局安装：npm install -g <package_name> (例如：npm install -g eslint)
    • 本地安装：npm install <package_name> (例如：npm install express)

  2. 更新包:

    • 更新所有包：npm update
    • 更新特定包：npm update <package_name> (例如：npm update express)

  3. 卸载包:

    • 卸载全局包：npm uninstall -g <package_name>
    • 卸载本地包：npm uninstall <package_name>

  4. 查看已安装的包:

    • 查看全局已安装的包：npm list -g --depth=0
    • 查看本地已安装的包：npm list --depth=0

  5. 初始化项目:

    • 创建一个新的 package.json 文件：npm init (按提示输入相关信息)
    • 创建一个空的 package.json 文件：npm init -y

  6. 运行脚本:

    • 在 package.json 中定义的脚本可以使用 npm run <script_name> 运行。例如：npm run build 或 npm start

  7. 查看帮助信息:

    • 查看 npm 的帮助信息：npm help

  8. 清理缓存:

    • 清理 npm 缓存：npm cache clean --force

  9. 查看版本信息:

    • 查看 npm 版本：npm -v
    • 查看 Node.js 版本：node -v

这些命令可以帮助你有效地管理和维护项目中的依赖关系。如果你有更具体的需求或者遇到问题，可以告诉我，我会尽力帮助你。

npm-修改源

修改 npm 源是为了提高包的下载速度，尤其是在国内网络环境下，使用国内的镜像源（如淘宝镜像源）可以显著提升下载速度。以下是几种常见的方法来修改 npm 的源：

方法一：使用 npm config 命令

  1. 设置淘宝镜像源

    npm config set registry  https://registry.npmmirror.com

  2. 验证是否设置成功

    npm config get registry

    如果输出为 https://registry.npm.taobao.org，则说明设置成功。

方法二：使用 nrm 工具

nrm 是一个 npm 源管理工具，可以方便地切换不同的 npm 源。

  1. 安装 nrm

    npm install -g nrm

  2. 列出可用的源

    nrm ls

  3. 切换到淘宝源

    nrm use taobao

方法三：使用 .npmrc 文件

你也可以直接在项目的根目录下创建一个 .npmrc 文件，并添加以下内容：

registry=https://registry.npm.taobao.org

这样，在该目录下的 npm 操作都会使用这个配置的源。

方法四：临时指定源

如果你只是想临时指定某个包的安装源，可以在安装命令后面加上 --registry 参数：

npm install <package-name> --registry=https://registry.npm.taobao.org

以上是几种常见的修改 npm 源的方法，你可以根据自己的需求选择适合的方式来加速 npm 包的下载。
